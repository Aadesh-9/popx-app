# PopX React UI

A simple React project that recreates the PopX mobile UI screens from the provided Adobe XD design.

This project includes:

- Welcome screen
- Login screen
- Create Account screen
- Account Details screen

The app is designed to look like a mobile app centered on a desktop browser, with smooth navigation between pages.
